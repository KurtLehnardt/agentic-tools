# OpenClaw Presentation USB

## What this USB contains

This drive is meant to be both:
- a **human-friendly handoff** after the talk
- an **agent-friendly starter pack** you can feed directly to your own OpenClaw workspace

## Suggested use

1. Read the Markdown guides directly
2. Open the diagrams while following the deck
3. Copy the templates into your own OpenClaw workspace
4. Edit them until they reflect your environment, not ours
5. Start with one healthy agent, not an impressive architecture
